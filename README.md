## вҡЎ What is ETFSignal AI?

Most crypto traders are drowning in noise. BTC/ETH spot ETF flows, institutional inflows, on-chain events, and market news fire constantly вҖ” but no single tool **connects it all**, **explains what it means**, and **lets you act on it**.

**ETFSignal AI** closes that gap.

It pulls institutional ETF flow data from SoSoValue, computes a live sentiment score, synthesizes everything with Claude AI into a clear market signal, then lets you execute the trade directly on SoDEX testnet вҖ” all from one dashboard with 5 focused views.

## рҹҺҜ The Full Pipeline

| Step | What Happens |
|------|-------------|
| **1. Ingest** | Fetches live BTC & ETH spot ETF metrics вҖ” daily inflows, net assets, cumulative flows, per-fund breakdown |
| **2. Sentiment** | Computes a 0вҖ“100 flow sentiment score from 14-day momentum + inflow breadth |
| **3. Enrich** | Pulls categorised news: on-chain events, institutional reports, price alerts, research |
| **4. Analyze** | Claude AI synthesises everything into a structured signal: direction, confidence, key factors, risk warning |
| **5. Signal** | Dashboard shows BULLISH / BEARISH / NEUTRAL with headline, summary, trade idea, AskAI follow-up chat |
| **6. Execute** | User connects MetaMask, reviews order, acknowledges risk, signs via EIP712, submits to SoDEX testnet |

## рҹ–ҘпёҸ App Pages

The app is split into 5 focused views, all sharing one live data context (no re-fetch on navigation):

| Route | Purpose |
|-------|---------|
| `/app` | **Overview** вҖ” full cockpit: ETF panels, price/flow chart, donut, sentiment gauge, signal, alerts, news |
| `/app/flows` | **Flows** вҖ” BTC + ETH inflow charts and market-share donuts side-by-side |
| `/app/signals` | **Signals** вҖ” sentiment gauge, AI signal, AskAI chat, signal archive, how-it-works explainer |
| `/app/alerts` | **Alerts** вҖ” filterable grid of smart flow & anomaly alerts (by type + severity) |
| `/app/news` | **News** вҖ” searchable news grid with category filter |

## рҹҡҖ Quick Start

### Prerequisites

- Node.js 18+ / pnpm 9+
- MetaMask browser extension
- SoSoValue API key ([apply here](https://forms.gle/2nuJT2qNbUQsyyZy8)) вҖ” *app works in demo mode without one*
- Anthropic API key ([get here](https://console.anthropic.com))

### 1. Clone & Install

```bash
git clone https://github.com/girume1/etfsignal.git
cd etfsignal/app
pnpm install
```

### 2. Configure Environment

```bash
cp .env.example .env
```

```env
# .env

# SERVER-SIDE вҖ” used only by /api/analyze edge function (never sent to browser)
ANTHROPIC_API_KEY=sk-ant-...

# CLIENT-SIDE вҖ” leave as placeholder to run in demo/mock mode
VITE_SOSOVALUE_API_KEY=your_sosovalue_api_key_here
```

> **Security note:** `ANTHROPIC_API_KEY` has **no** `VITE_` prefix вҖ” it is a server-only variable consumed by the Vercel Edge Function `/api/analyze`. It is never bundled into the browser build.

### 3. Run

```bash
pnpm dev
```

Open [http://localhost:5173](http://localhost:5173) рҹҺү

### Mock Mode

When `VITE_SOSOVALUE_API_KEY` is empty or a placeholder, all fetchers return from `mockData.ts`:
- 10 BTC funds (IBIT, FBTC, ARKB, BITB, HODL, BRRR, BTCO, EZBC, BTC, GBTC)
- 9 ETH funds (ETHA, FETH, ETHW, CETH, ETHV, QETH, EZET, ETH, ETHE)
- 14-day inflow series + price history per asset
- 5 smart alerts + 4 historical signals with mock P&L
- 12 realistic news headlines

If a real key returns 401, the session silently degrades to mocks вҖ” no crash, no empty UI.

### Signal Output Schema

```typescript
interface MarketSignal {
  direction:   'BULLISH' | 'BEARISH' | 'NEUTRAL';
  confidence:  number;        // 0вҖ“100
  headline:    string;        // One punchy sentence
  summary:     string;        // 2вҖ“3 sentences with specific numbers
  keyFactors:  string[];      // Top 3 drivers
  tradeIdea:   string;        // Actionable suggestion
  riskWarning: string;        // Main risk to the thesis
  timestamp:   Date;
}
```

## рҹӣЎпёҸ Safety & Risk Controls

- **Explicit risk acknowledgment** вҖ” checkbox required before any trade executes
- **Risk warning displayed** вҖ” Claude's risk thesis shown prominently before confirmation
- **Confidence score** вҖ” visual bar shows AI certainty (0вҖ“100%)
- **Testnet-only** вҖ” all trades go to SoDEX testnet (`chainId: 138565`), no real funds
- **"Not financial advice"** вҖ” displayed on every signal card and trade modal
- **Trade review step** вҖ” full order details visible (symbol, side, quantity) before signing
- **API key isolation** вҖ” Anthropic key lives only in Vercel Edge Function, never in the browser bundle
